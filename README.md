# 二補碼反轉計算機 (Two's Complement Calculator)

一個簡單的網頁工具，幫助理解二補碼 (Two's Complement) 的計算過程。

## 功能

- 支援 **8-bit** 與 **16-bit** 位元長度
- 可輸入 **十進制** 或 **二進制** 數值
- 逐步展示計算過程：
  1. 原始數值的二進制表示
  2. 一補碼 (1's Complement)
  3. 二補碼 (2's Complement)
- 輸入範圍自動檢查（符合有號二補碼範圍）

## 使用方式

直接開啟 `index.html` 即可使用，無需任何依賴或伺服器。
